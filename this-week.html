      How this works

      By default this shows a digest with summaries from data/news.json (refreshed weekly by a GitHub Action that pulls RSS feeds — each item carries a 1–2 line summary). Click Refresh to pull live headlines from the public Hacker News search API (titles ranked by points, last 7 days). Offline, a curated set of briefings with summaries is shown.

      Tip: titles link to the source article; discuss opens the Hacker News thread.
